错误处理

  • try catch finally
    • finally中的内容即使return依然会运行到

单线程

  • JavaScript是单线程的
  • 使用异步回调来增加异步任务的性能
  • Event Loop
    • 执行栈 + 任务队列，当栈空的时候会去读取任务队列。
  • Nodejs的process.nextTick阮一峰

  • Web Worker(Html5的多线程，作为主线程的子线程)

    • 执行的脚本必须来自网络
    • 通信必须通过消息(postMessage发送，onmessage接收)
    • 不能使用主线程的DOM对象
