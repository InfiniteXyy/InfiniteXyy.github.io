面试题（一）搜索

2019年3月16日

人是擅长归纳总结的，当你重复一件事情多次以后，自然而然地就能找到其中的规律。于是按照以下规则来做题就可以事半功倍。

  1. 优先完成相似题目
  2. 从Easy的题目开始
  3. 思考过后仍做不出来就看答案
  4. 对于曾看答案的题目，要重复做
  5. 做完相似题目，对其归纳总结

注1：这些方法都是来自于花花酱的一期视频，相似题目来自于LeetCode 题目分类(感谢huahua)，大家可以参考了刷题~

注2：这篇博客只是一份个人笔记，如有错误，还请指出啦，谢谢~
