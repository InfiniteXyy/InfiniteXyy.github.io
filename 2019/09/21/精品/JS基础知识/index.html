数据类型

typeof：判断基本数据类型(null 也是 object)
instanceof：判断具体是哪种对象
number object string boolean undefined null symbol

  • in操作符与hasOwnProperty
  • Map、Set
  • 解构：默认值，多层解构，swap

控制流

  • interable,forEach,for…of…

函数

  • 变量作用域，var 互不影响、从内而外
  • 变量提升(可以访问，但是是undefined)
  • window 全局作用域(NodeJS 是 global)
  • this与 strict mode 与执行栈 知乎 ◦ this 绑定(对象或全局)
    • 词法环境(变量映射、外部此法环境的引用)
    • 全局环境
    • 函数环境(arguments 对象）
    • 变量环境(属于词法环境，经用于存储 var 的绑定)
  • apply/call(参数的不同) Github
  • 箭头函数的词法作用域
  • 生成器(yield) 与 ajax
  • 闭包：函数与函数中的变量 知乎 * 如何实现？堆

面向对象

  • 基本对象 Date_Reg_JSON
    • Date 在 JS 中月份从 0 开始
    • Reg 用于 split 很方便
    • Reg 还可以用来分组(使用括号)
    • JSON.stringfy(object, replacer, space)
  • this，与运行上下文
  • 原型链，一个对象的原型指向另一个对象
  • 为原型增加方法，使用 prototype
  • prototype vs__proto__ 知乎
    • prototype 是函数特有的对象，它只有一个属性 constructor，它的内容是函数自身
    • proto 指向构造它的函数的 prototype
  • class

错误处理

  • try catch finally * finally 中的内容即使 return 依然会运行到
  • 若抛出错误，会一直向上寻找 catch，直到 JavaScript 引擎
  • 异步操作需要在回调函数中捕获异常

单线程

  • JavaScript 是单线程的
  • 使用异步回调来增加异步任务的性能
  • Event Loop * 执行栈 + 任务队列，当栈空的时候会去读取任务队列。
  • NodeJS 的process.nextTick 阮一峰
  • Web Worker(Html5 的多线程，作为主线程的子线程)
    • 执行的脚本必须来自网络
    • 通信必须通过消息(postMessage 发送，onMessage 接收) * 不能使用主线程的 DOM 对象

Node.JS

  • 模块
    • module.exports 与 exports(后者只是前者最初的一个引用，最后看的还是 module.exports,好苦逼啊) 掘金
    • 模块的原理(闭包)
  • Fs(readFile, readFileSync)
  • stream
  • crypto(md5、AES、RSA……)

其它
